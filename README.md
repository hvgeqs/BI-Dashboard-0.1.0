# How to start
Make sure you have [node.js](https://nodejs.org/en) installed

Install dependencies and start project on the localhost
```
 cd .\BI-Dashboard-0.1.1-main
```
```
 npm init -y
```
```
 npm install
```
```
 npm start
```
